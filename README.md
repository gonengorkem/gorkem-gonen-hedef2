# Görkem Gönen Hedef

## Proje Amacı
Bu proje, Gelir İdaresi Başkanlığı'nın (GİB) e-Dönüşüm (e-Fatura, e-Arşiv, e-Müstahsil Makbuzu, e-İrsaliye, ve e-Serbest Meslek Makbuzu) süreçleri için yayınladığı güncel XML paketlerindeki şema ve şematron değişikliklerini otomatik olarak tespit edip kontrol edecek bir sistemin geliştirilmesini hedeflemektedir.

## Temel İhtiyaçlar
* GİB tarafından yayınlanan yeni paketlerin (XML formatında) okunması.
* Mevcut sistemdeki kontrollerle yeni paketteki kuralların/şemaların karşılaştırılması.
* Yeni eklenen/çıkarılan veya değiştirilen kuralların (şematron vb.) tespit edilmesi.
